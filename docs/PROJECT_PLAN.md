# Sales Analytics Project Plan

## Project Overview

This project aims to analyze retail sales data using PostgreSQL, Microsoft Excel, and Power BI. The objective is to transform raw sales data into meaningful business insights through data cleaning, SQL analysis, and interactive dashboards.

---

## Project Objectives

- Clean and prepare the dataset for analysis.
- Analyze sales performance across different regions and product categories.
- Identify the most profitable products and customers.
- Evaluate the impact of discounts on profit.
- Build interactive dashboards using Excel and Power BI.
- Present actionable business insights and recommendations.

---

## Business Questions

1. Which product categories generate the highest sales?
2. Which products generate the highest profit?
3. Which regions perform the best and the worst?
4. How do discounts affect profitability?
5. Who are the top customers by sales?
6. What are the monthly sales trends?
7. Which states have the lowest profitability?
8. Which shipping mode is used most frequently?
